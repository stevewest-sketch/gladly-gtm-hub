# Gladly Design System Skill

This skill ensures all UI/UX updates maintain consistency with the established Gladly demo design patterns.

## Core Design Principles

1. **Professional and Clean**: Minimalist approach with generous whitespace
2. **Brand Consistency**: Green primary (#009B00) for actions and success
3. **Clear Hierarchy**: Well-defined typography scale and color usage
4. **Accessibility First**: Ring focus states, semantic HTML, ARIA labels
5. **Performance**: Subtle animations, 200ms transitions

## When to Use This Skill

Load this skill when:
- Making UI/design changes to the GTM Hub
- Creating new components or pages
- Updating existing components
- Working with colors, typography, or spacing
- Need to reference Gladly's design standards

## Quick Reference

**Primary Colors:**
- Brand Green: `#009B00` (actions, success)
- Brand Purple: `#8C69F0` (accents, badges)
- Neutral 900: `#0D0D0D` (primary text)
- Neutral 700: `#252525` (headers, nav)
- Neutral 200: `#DFDFDF` (borders)
- Neutral 100: `#F3F3F3` (backgrounds)

**Typography:**
- Section headers: `text-[18px] font-semibold`
- Body text: `text-[15px] font-normal`
- Labels: `text-[12px] font-semibold`

**Spacing:**
- Base unit: 4px
- Section padding: `p-4` or `p-6`
- Element gaps: `gap-4`
- Borders: `border-[var(--neutral-200)]`

For complete guidelines, see the full documentation in `.claude/skills/gladly-design.md`.
